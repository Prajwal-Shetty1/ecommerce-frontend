## 📌 About The Project

This is the **user panel** of the e-commerce application.
Users can browse products, add items to cart, and place orders.

---

## ✨ Features

* User register / login / logout
* Browse products and view details
* Add to cart and manage quantity
* Advanced product filtering and sorting
* Select product size
* Place orders with multiple payment methods
* View order history

---

## 🤖 AI Fashion Assistant

**Forever Personal Stylist** helps users make better fashion decisions by providing personalized styling suggestions and outfit recommendations.
